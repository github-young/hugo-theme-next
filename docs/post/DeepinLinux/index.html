体验Deepin Linux

时间： 2019-01-05   |   分类： Tech     |   阅读

看到V站的帖子，以前只在虚拟机里装过，于是决定自己实机体验一下。

安装

用了deepin自己的启动盘制作工具，不过装的时候发现竟然没有live环节直接安装，要想体验live还需要专门下载另一个iso文件。Intel 545s，安装大概花了20分钟。

体验

进入系统感觉很漂亮，其次一个惊喜就是ubuntu上死活装不了的指纹识别竟然是开箱即用。当然最有吸引力的就是deepin的qq、微信等wine软件，使用体验不错，特别感谢deepin的努力解决了这个痛点。

当然，问题总是有的。我发现我在win10和ubuntu1804上使用正常的 Fn+左右方向键 在这里就不行，但是其它的 Fn+功能键 是可以的。最终没能完美解决，我把 Caps_Lock+方向键 映射成了 Home 和 End 。另外，在别的系统可以校园网接网线直接访问谷歌，这里也不行，折腾了两天，最后发现换成tuna的dns就解决了。

总结

在Linux的范围里，从生产的角度，deepin不一定是最好的，但是从生活的角度，对于中国人我觉得是最好的。我准备放假给父母试用一下。

后记

回家给父母试了，以没有Office被一票否决，1551…

后后记

Manjaro+DDE 天下第一！
