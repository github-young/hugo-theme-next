ClipDict: Clipboard Dictionary

时间： 2020-01-09   |   分类： Tech     |   阅读

ClipDict – 从剪贴板取词的命令行字典。

演示

使用

下载 ClipDict 到本地，运行它即可：./ClipDict.sh。

ClipDict 只是一个shell脚本，它依赖如下两个软件：

  • xclip
  • dictd

请先安装它们：

1

sudo pacman -S xclip dictd

注意：dictd只是一个字典软件，不自带词典。需要自己下载和配置本地中文词典。示例使用的是朗道中英文词典(langdao-ec)。【详情待补充】
