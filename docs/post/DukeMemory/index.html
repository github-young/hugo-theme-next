Duke Memory

时间： 2017-12-27   |   分类： Duke     |   阅读

杜克项目结束了，把能想到的一些记在这里。

很久以前的几句话：

  • 一定要选Independent Study，不然导师很难找(欢迎明年的同学找FEL的Wu教授)
  • 暑假一定要预习量子力学(教材：C-TDL)
  • 记得带指甲剪和剪刀
  • 一位不愿透露姓名的群主：如果你很浪，那一点都不愁foodpoints花不完
  • zcy：一定要选五星级酒店，300swift宿舍(我：在中校区)

新加的(大概按时间顺序)：

关于电话卡(7月)

往届同学都用的是中国电信美洲公司的电话卡，这张卡在安卓手机上的使用体验极差。基本上只能打电话和发短信，流量几乎无法使用，因为根本没有信号。奇怪的是在苹果手机上可以正常使用。

结论就是如果你在美国非常需要稳定可靠的流量，请不要使用它。当然，Duke里面都有WiFi，如果不出去旅游，使用它应该还可以。毕竟还有每人10刀的优惠(充值的时候使用优惠码，双方都可以加10刀话费)。比如我的推荐码深入人心，所以我一共获得了90刀的话费，一学期没充钱嘿嘿嘿。

除了中国电信美洲公司的卡，还可以办美国运营商的，比如bz就办了AT&T的，使用体验当然十分好，就是贵了点。

关于银行卡(8月底)

到美国，一张当地银行卡可以说是必备的，Duke周围比较好办的就三家：BoA, Suntrust, Wells Fargo. 其中Suntrust距离西校区最近，因此这一届大部分同学都办的是Suntrust，还有少数几位同学办的是BoA，而Wells Fargo在东校区附近，今年没有人办。

经过一学期的切身体会，大家一致觉得BoA的银行卡最好。理由如下：Suntrust有“月费”3刀，BoA没有；BoA是大银行而Suntrust是小银行，去纽约旅游的时候，路上就没找到Suntrust的ATM机；给学校账单付款的时候，Suntrust失败过两次(LHZ同学)，最后使用BoA成功。

当然，BoA有一个缺点，它的支行离学校比较远，要打车去。

PS：在没办当地银行卡前，确实可以先用国内办的VISA卡用，甚至南大的工行卡也可以直接在ATM机上取出美元，但是建议只在大公司和校内机构用，因为据说国内的卡在美国被盗刷会很麻烦。等办了美国银行卡，就尽量用美国的卡，这样即使盗刷，处理起来会简单得多。

关于采购物资(8月底)

离Duke最近的超市应该就是大约10分钟车程的Target。开学初肯定要去一次的，买齐生活用品，毕竟不像国内买东西这么方便。当然Duke里有校园超市Duke Store，就是比较贵，偶尔也会买点东西。开学前学校会组织一次免费的往返Target的活动，建议那个时候去，我去年错过了就打uber去的。

基本上去Target买了各种生活用品，但有几个坑注意一下：台灯和电热水壶完全没必要买。我一开始到美国面对没有热水的生活确实不放心，结果买了热水壶就没用过(相信我，你们很快就能习惯早起喝一大口冷水的生活)。洗衣液都是2L这样的大桶，建议一栋楼的几个人一起买共用，我到学期末连三分之一都没用完。其它的东西自己到那应该心里有点数的。

Target旁边就是黎明电推华人超市，可以买到看上去十分亲切的东西(没找到辣条)。基本上我们把它当做菜市场，买肉买菜买铁锅等等。

关于不务正业(8~12月)

Duke宿舍里的commons room真是比南大不知道高到哪里去好太多了。这就是表白墙常见的“可以大声讲话、自由活动、又不影响别人的封闭室内空间”。

Duke给每个学生提供了3D打印的资源，大家可以薅资本主义的羊毛提高动手能力。我看到陈列区有人打印了Pickle Rick和刺客信条里的袖剑。关于3D打印的使用方法，可以看我的另一篇文章。
